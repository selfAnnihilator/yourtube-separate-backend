# Reports flag comments for review

Reported Comments remain visible and are Flagged for Review instead of being automatically hidden or deleted by report counts or dislike counts. This avoids letting coordinated reports or dislikes suppress comments, while still preserving moderation signals for a future review workflow.

## Considered Options

- **Auto-delete by dislike threshold**: rejected because Comment Reactions are engagement feedback, not moderation.
- **Auto-hide by report count**: rejected because reports can be abused to suppress visible comments before review.
- **Flag for review**: accepted because it separates user safety signals from destructive moderation actions.
